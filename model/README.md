# Model — Resonate ML & data

Python/uv project for training, data prep, and seeding. All ML work lives here.

## Setup

From this directory:

```bash
uv sync
```

This creates `.venv` and installs dependencies from `pyproject.toml`.

## Layout

- **scripts/** — Seed DB, export mappings, show sample data
- **notebooks/** — Jupyter: data exploration, dataset build, training
- **data/** — Processed data (CSVs, npz). Git-ignored; generated by notebooks/scripts.
- **dataset/** — Raw Last.fm dataset. Git-ignored.

## Running scripts

Use the repo root `.env` for `DATABASE_URL` (same as the SvelteKit app). Run from `model/`:

```bash
cd model
uv run python scripts/seed_db.py [--reset]
uv run python scripts/seed_user_profiles.py
uv run python scripts/export_mappings.py
uv run python scripts/show_sample_data.py
```

## Notebooks

Start Jupyter from `model/` so paths resolve correctly:

```bash
cd model
uv run jupyter notebook
# or: uv run jupyter lab
```
